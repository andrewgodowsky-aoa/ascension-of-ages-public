# Ascension of Ages

Ascension of Ages is a NeoForge 1.21.1 Minecraft modpack project focused on age-based progression, FTB Quests structure, staged unlocks, KubeJS automation, and curated long-form progression.

This repository tracks the authored source for the pack. It intentionally does not redistribute downloaded mod jars, local worlds, logs, caches, launcher state, or third-party resource/shader packs.

## What Belongs Here

- Pack configuration and defaults
- FTB Quests files and language files
- KubeJS scripts, custom recipes, and custom items
- AStages / KubeJS gating configuration
- Datapacks and authored pack data
- Release notes and contribution guidance
- Lightweight helper scripts for maintenance

## What Does Not Belong Here

- `mods/`
- `saves/`
- `logs/`
- `resourcepacks/` and `shaderpacks/`
- CurseForge launcher caches and generated instance state
- Export zips, server pack zips, and other generated binaries

## Update Flow

Use `CHANGELOG.md` for player-facing changes and `CONTRIBUTING.md` for community contribution guidance.
